D-Town Academy


" D-Town Academy is a E-Learning web application designed to offer students with seamless access to an array of courses addedd by trainers. Within this platform, students have the opportunity to engage actively by leaving comments, providing valuable feedback, and fostering a collaborative learning environment. Additionally, students have the flexibility to explore both free and paid courses, making learning accessible to all."


• Led the development of a e-learning web application, bridging the gap between educators and learners, resulting in a successful platform serving 20 active users till now.


• Utilized a tech stack including Java Spring Boot, Thymeleaf, HTML, CSS, Bootstrap, JavaScript, Hibernate, and JDBC to create a user-friendly and responsive platform.


• Designed and implemented RESTful APIs and microservices for scalable course management with secure Razorpay transactions


• Hosted on Heroku cloud with PostgreSQL for data and secured "vivek.dtownacademy.online" domain from Hostinger.


• Designed user interfaces for Students, Trainers, and Administrators, streamlining course management, registration, and transactions, with Admin interface for full control
